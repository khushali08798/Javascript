# Ashwin Mishra - Portfolio Website

A modern, responsive professional portfolio website built with HTML5, CSS3, and Vanilla JavaScript.

## Features

- **Responsive Design**: Fully responsive for desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean design with smooth animations and professional color palette
- **Sticky Navigation**: Fixed navigation bar with mobile hamburger menu
- **Form Validation**: Client-side form validation with real-time feedback
- **SEO Optimized**: Proper meta tags, semantic HTML, and accessibility features
- **Smooth Scrolling**: Smooth scroll behavior for navigation links
- **Card-based Layout**: Modern card layouts for projects and highlights

## Project Structure

```
portfolio-website/
├── index.html          # Home page with hero section and highlights
├── about.html          # About page with professional summary
├── experience.html     # Experience timeline
├── projects.html       # Projects showcase with skills section
├── contact.html        # Contact form and information
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── script.js       # JavaScript for interactions and validation
└── images/             # Image assets (profile.jpg, project images, etc.)
```

## Setup Instructions

1. Clone or download the project files
2. Place the website files in your web server directory
3. Add your profile image as `images/profile.jpg` (recommended size: 300x300px)
4. Customize the content in the HTML files to match your information
5. Update contact information in `contact.html`
6. Modify colors and styling in `css/style.css` if needed

## Customization

### Content Updates
- Update personal information in all HTML files
- Modify experience details in `experience.html`
- Add/remove projects in `projects.html`
- Update contact information in `contact.html`

### Styling Changes
- Colors: Update CSS custom properties in `css/style.css`
- Fonts: Change font-family in the CSS
- Layout: Modify grid layouts and spacing

### Adding Images
- Profile image: `images/profile.jpg`
- Project screenshots: `images/project1.jpg`, etc.
- Background images: Add to `images/` folder and reference in CSS
